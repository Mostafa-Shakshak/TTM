# TTM – Talk To Me

TTM (Talk To Me) is a full-stack social media platform that combines social networking with real-time messaging. The project focuses on building a scalable backend architecture, modern responsive UI, and a smooth user experience.

✨ Features

### Authentication
- User Registration & Login
- JWT Authentication
- Refresh Token
- Password Reset via OTP
- Secure Password Hashing (bcrypt)

### Social Features
- Create, Edit and Delete Posts
- Like & Comment System
- Share Posts
- Search Users & Posts
- Public & Private Profiles
- Follow / Unfollow
- Follow Requests
- Notifications

### Messaging
- Private Chats
- Group Chats
- Add / Remove Members
- Promote & Demote Members
- Leave Groups
- Archive / Unarchive Conversations
- Delete Conversations
- Reply to Messages
- Edit Messages
- Delete Messages
- Real-time Messaging
- Online Status
- Typing Indicator
- Delivered & Seen Status

### Media
- Profile Picture Upload
- Cover Image Upload
- Image Messages
- Cloudinary Integration

### UI / UX
- Fully Responsive Design
- Mobile & Desktop Support
- Modern User Interface
- Dark Theme
- Smooth User Experience
